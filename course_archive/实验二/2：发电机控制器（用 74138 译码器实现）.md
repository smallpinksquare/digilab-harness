## 2. 逻辑表达式

$$X = \sum m(1,2,3,4,7)$$
$$Y = \sum m(5,6,7)$$

## 3. 转换为 74138 的固定格式

为避免与控制输出变量 `Y` 混淆，下面把 **74138 的输出引脚**记为 `D0~D7`（低有效）：

- `Dk = 0` 当且仅当最小项 `m_k = 1`
- 即 `m_k = \overline{D_k}`

因此：
$$\sum m_i = \overline{\overline{m_i}\cdot\overline{m_j}\cdots} = \overline{D_i\cdot D_j\cdots}$$

即：**最小项之和 = 对应译码器输出端取与非**。

$$\boxed{X = \overline{D_1\cdot D_2\cdot D_3\cdot D_4\cdot D_7}}$$

由于 `X` 含 5 个最小项，其反函数 `\overline{X}=\sum m(0,5,6)` 只有 3 个最小项，更省门：
$$\overline{X} = \overline{D_0\cdot D_5\cdot D_6}$$
$$X = \overline{\overline{X}} = \overline{\overline{D_0\cdot D_5\cdot D_6}}$$

对应门级实现（更直接）：
- `\overline{X} = NAND4(D0, D5, D6, 1)`（用 4 输入与非替代 3 输入）
- `X = NAND2(\overline{X}, \overline{X})`

控制输出 `Y`（发电机 Y 机组）为：
$$\boxed{Y = \overline{D_5\cdot D_6\cdot D_7}}$$
即可实现为 `Y = NAND4(D5, D6, D7, 1)`。
