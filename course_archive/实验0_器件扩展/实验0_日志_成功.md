## 算法思路

按工单四步执行：

1. **Step 1（设计确认）**：确定 `Primitive + Block + units` 扩展路径，保持对 7400/7420 完全兼容。
2. **Step 2（74138 落地）**：实现 `DECODE3`、多 LHS 解构赋值、Block 求值链路。
3. **Step 3（74153 + 混用）**：实现 `MUX4`、注册 74153，增加跨芯片混用示例并跑通。
4. **Step 4（本文档收口）**：补齐附录/README/CHANGELOG，做最终回归与工单关闭。

核心策略：

- 解析层：原语输出数表驱动（`_PRIMITIVE_OUTPUT_COUNT`）+ 多 LHS 强约束。
- 综合层：`alloc_gate`（门）与 `alloc_block`（块）并行分配。
- 验证层：统一 `units` 抽象（单输出 gate 与多输出 block 同拓扑求值）。
- 物理优化：保留菊花链扇出与 VCC 输入替换，对真值求解透明。

```

74153_A.7 -> 7400_A.1

74153_A.9 -> NC.0
74153_A.10 -> NC.0
74153_A.11 -> NC.0
74153_A.12 -> NC.0
74153_A.13 -> NC.0
74153_A.15 -> NC.0
74153_A.16 -> VCC.0
```

门/块映射（摘要）：

- `74138_A.block0` 先解码 `C,B,A -> Y0..Y7`
- `74153_A.block0` 选通 `Y0..Y3` 生成 `M`
- `7400_A.gate0` 做自反 NAND：`F = NAND2(M, M) = NOT(M) = NOT(C)`
