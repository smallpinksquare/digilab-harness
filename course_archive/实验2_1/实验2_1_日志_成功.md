## 算法思路

目标函数：

`F = (¬A1 + B1)(¬A0 + B0)`

74151 是 8 选 1 数据选择器，函数形式 `Y = D[ 4·C + 2·B + A ]`（使能有效时）。取 select `S2 S1 S0 = A1, A0, B1`，把 4 变量真值表按 `(A1A0B1)` 分组（每组 2 行，B0 翻转），对照得到 8 个数据端：

| 选择 (A1 A0 B1) | F(B0=0) | F(B0=1) | Di |
|:---:|:---:|:---:|:---:|
| 000 | 1 | 1 | D0 = 1  |
| 001 | 1 | 1 | D1 = 1  |
| 010 | 0 | 1 | D2 = B0 |
| 011 | 0 | 1 | D3 = B0 |
| 100 | 0 | 0 | D4 = 0  |
| 101 | 1 | 1 | D5 = 1  |
| 110 | 0 | 0 | D6 = 0  |
| 111 | 0 | 1 | D7 = B0 |

8 个数据端均为常量 0/1 或变量 B0，**不需要任何 NAND 门**。最终表达式：

```txt
F, F_BAR = MUX8(A1, A0, B1, 1, 1, B0, B0, 0, 1, 0, B0)
```

> `F_BAR` 对应 74151 pin 6（Ȳ），物理上保持悬空（或可选作监测探针）；本电路只用 Y（pin 5）。

## 完整电路连接方式（按附录 B.4 排序）

来自 [circuit.txt](circuit.txt)，每行格式 `源.引脚 -> 目的.引脚`。本电路应用了菊花链扇出：同一恒定信号（VCC/GND）或同一变量信号（B0）的多个 D 输入串联在一起，物理上对应"从一个 pin 用短跳线串到下一个 pin"的接法。

```
GND.0 -> 74151_A.13
INPUT.A0 -> 74151_A.10
INPUT.A1 -> 74151_A.9
INPUT.B0 -> 74151_A.1
INPUT.B1 -> 74151_A.11
VCC.0 -> 74151_A.3
74151_A.1 -> 74151_A.2
74151_A.2 -> 74151_A.12
74151_A.3 -> 74151_A.4
74151_A.4 -> 74151_A.14
74151_A.5 -> OUTPUT.F
74151_A.7 -> GND.0
74151_A.8 -> GND.0
74151_A.13 -> 74151_A.15
74151_A.16 -> VCC.0
```

### 数据端 ↔ 物理引脚映射

| 数据端 | 引脚 | 应接信号 | 物理来源 |
|:------:|:----:|:--------:|:--------:|
| D0 | pin 4  | 1   | 菊花链 VCC（共享 D1 那条 VCC 链）|
| D1 | pin 3  | 1   | VCC.0 → pin 3 |
| D2 | pin 2  | B0  | 菊花链 B0（共享 D3 那条线）|
| D3 | pin 1  | B0  | INPUT.B0 → pin 1 |
| D4 | pin 15 | 0   | 菊花链 GND（共享 D6 那条 GND 链）|
| D5 | pin 14 | 1   | 菊花链 VCC（共享 D1 那条 VCC 链）|
| D6 | pin 13 | 0   | GND.0 → pin 13 |
| D7 | pin 12 | B0  | 菊花链 B0（共享 D3 那条线）|

### 菊花链摘要

| 信号 | 物理走线 |
|------|----------|
| `B0`（D3, D2, D7）| `INPUT.B0 → pin 1 → pin 2 → pin 12` |
| `1`（D1, D0, D5）| `VCC.0 → pin 3 → pin 4 → pin 14` |
| `0`（D6, D4）| `GND.0 → pin 13 → pin 15` |
