# 跨芯片混用示例（7400 + 74138 + 74153）

目标：构造一个同时使用三种芯片的最小可验证电路。

```txt
chips:   7400 x 1, 74138 x 1, 74153 x 1
inputs:  C, B, A
outputs: F

Y0, Y1, Y2, Y3, Y4, Y5, Y6, Y7 = DECODE3(C, B, A)
M = MUX4(B, A, Y0, Y1, Y2, Y3)
F = NAND2(M, M)
```

推导：

- `DECODE3(C,B,A)`：当 `C=0` 时，`Y0..Y3` 中被选一路为 0；当 `C=1` 时，`Y0..Y3` 全为 1
- `M = MUX4(B,A,Y0,Y1,Y2,Y3)` 因此等价于 `M = C`
- `F = NAND2(M,M) = NOT(M) = NOT(C)`
