# 实验 2_1：血型配对（74151）

目标函数：

`F = (¬A1 + B1)(¬A0 + B0)`

其中 `A1A0` 为输血者血型编码，`B1B0` 为受血者血型编码。

## 转换为 74151 的固定格式

74151 是 8 选 1 数据选择器，函数形式为：

`Y = D[ 4·C + 2·B + A ]`（使能有效时）

取选择端 `S2 S1 S0 = A1, A0, B1`，把 4 变量真值表按 `(A1 A0 B1)` 分组（每组 2 行，B0 翻转）。对照得：

| 选择 (A1 A0 B1) | F(B0=0) | F(B0=1) | 数据端 Di |
|:---:|:---:|:---:|:---:|
| 000 | 1 | 1 | D0 = 1  |
| 001 | 1 | 1 | D1 = 1  |
| 010 | 0 | 1 | D2 = B0 |
| 011 | 0 | 1 | D3 = B0 |
| 100 | 0 | 0 | D4 = 0  |
| 101 | 1 | 1 | D5 = 1  |
| 110 | 0 | 0 | D6 = 0  |
| 111 | 0 | 1 | D7 = B0 |

8 个数据端均为常量 0/1 或变量 B0，不需要任何 NAND 门。器件清单：单片 74151。

```txt
chips:   74151 x 1
inputs:  A1, A0, B1, B0
outputs: F

F, F_BAR = MUX8(A1, A0, B1, 1, 1, B0, B0, 0, 1, 0, B0)
```

> `F_BAR` 对应 74151 pin 6（Ȳ），物理上保持悬空（或可选作监测探针）；本电路只用 Y（pin 5）。
