## 算法思路

按实验定义：

- `X = Σm(1,2,3,4,7)`
- `Y = Σm(5,6,7)`

令 `D0~D7` 为 74138 的低有效译码输出（`m_k = ¬Dk`）：

- `X_BAR = Σm(0,5,6) = ¬(D0·D5·D6) = NAND4(D0, D5, D6, 1)`
- `X = ¬X_BAR = NAND2(X_BAR, X_BAR)`
- `Y = ¬(D5·D6·D7) = NAND4(D5, D6, D7, 1)`

最终表达式：

```txt
D0, D1, D2, D3, D4, D5, D6, D7 = DECODE3(A, B, C)
X_BAR = NAND4(D0, D5, D6, 1)
X = NAND2(X_BAR, X_BAR)
Y = NAND4(D5, D6, D7, 1)
```
