# course_archive

本目录保存了项目最初作为数字电路实验课程辅助工具时积累的**课程专用产物**，不属于 `digilab` 库的核心代码。

---

## 内容说明

| 路径 | 内容 |
|------|------|
| `实验1_1/` | BCD 码数值判别电路（7400 + 7420）产物 |
| `实验1_2/` | 4 位二进制数值判别电路产物 |
| `实验2_1/` | 血型配对电路（74151）产物 |
| `实验2_2/` | 发电机控制器（74138）产物 |
| `实验0_器件扩展/` | 74138 / 74153 / 74151 器件扩展日志 |
| `实验一/` | 实验一原始需求文档 |
| `实验二/` | 实验二原始需求文档（含逻辑推导） |
| `真值表与逻辑表达式/` | 每个实验的 `*_表达式.md` + `*_真值表.md` 原始输入 |
| `示例_74138/` `示例_74153/` `示例_混用/` | 综合器示例产物 |
| `模拟电路程序harness.md` | 项目原始规范文档 |
| `harness附录.md` | 原始附录（含 MSI 扩展说明） |
| `更新工单.md` | 历史工单记录 |
| `数字逻辑设计实践B_教学计划_2026学年_V1.pdf` | 课程教学计划 |
| `run_all.py` / `run_all.ps1` | 一键回归脚本（重跑 synth + verify，校验 SHA256） |
| `重塑日志/` | 各 Phase 的独立 L2 审查报告 |

---

## 使用

```bash
# 从仓库根目录运行（需先 pip install -e .）
python course_archive/run_all.py
# 或
powershell -File course_archive/run_all.ps1
```

所有产物（`circuit.txt`、`verify_report.json`）应与迁移前的 SHA256 基线一致。

---

## 与 examples/ 的区别

- `course_archive/` 是课程历史产物，文件名为中文，路径中含实验编号。
- `examples/` 是面向开源受众的精简英文示例，被 `tests/test_examples_regression.py` 引用并纳入 CI。
